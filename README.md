# Job Profile Automation Framework 🚀

## 📌 Project Description
This project automates daily profile updates on a job portal to improve recruiter visibility.  
It performs login, navigates to the profile section, and uploads a resume automatically.

---

## 🛠 Tech Stack
- Java
- Selenium WebDriver
- Cucumber BDD
- TestNG
- Maven
- Jenkins
- GitHub

---

## ⚙️ Features
- Automated login functionality
- Profile navigation
- Resume upload automation
- Screenshot capture after execution
- Jenkins scheduled execution (daily run)

---

## 📂 Framework Design
- Page Object Model (POM)
- Reusable utility classes
- Cucumber BDD structure

---

## 🚀 How to Run
1. Clone the repository
2. Update credentials in code
3. Run TestRunner class

---

## 🔥 Key Highlights
- Real-time automation use case
- File upload handling using Selenium
- Jenkins cron job for daily execution
- Robust framework design
